Flash

플래시에 대한 기능입니다.

# isSupport

플래시가 설치되어 있는지 확인합니다. 
				exus.flash.isSupport();

# generate

플래시를 생성합니다. 
				exus.flash.generate(file, width, height, id);

# remove

플래시를 삭제합니다. 
				exus.flash.remove(id);

Speech

TTS 목소리를 출력합니다.

# speech

알림을 지원하는지 확인합니다. 
				exus.Speech.speech(word, speecher, pitch, rate);

Notify

알람창을 띄웁니다.

# isSupport

알림을 지원하는지 확인합니다. 
				exus.Notify.isSupport();

# getPermitLevel

권한 레벨을 확인합니다. 
				exus.Notify.getPermitLevel();

# getPermit

권한을 요청합니다. 
				exus.Notify.getPermit();

# Show

알람창을 띄웁니다. 
				exus.Notify.Show(message, icon, body, options);

Generator

조작할 수 있는 특정 배열을 얻습니다, 배열은 yield로 정의되어 차례차례대로 뽑을 수 있습니다.

# getNext

다음 배열값을 가져옵니다. 
				exus.Generator.getNext();

# getUniqueNum

유니크한 숫자 배열을 얻습니다. 
				exus.Generator.getUniqueNum(min, max, length);

# getUniqueStr

유니크한 문자 배열을 얻습니다. 
				exus.Generator.getUniqueStr(length);

Popup

팝업창에 대한 기능입니다.

# Enabled

팝업창 허용여부를 확인합니다. 
				exus.Popup.Enabled();

# Open

새로운 팝업창을 엽니다. 
				exus.Popup.Open(url, width, height, p_top, p_left);

OptionList

옵션값을 가지는 리스트에 대한 기능입니다.

# moveSelectedItem

선택된 아이템을 다른 리스트로 이동시킵니다. 
				exus.OptionList.moveSelectedItem(id, id);

# moveAllItem

모든 아이템을 다른 리스트로 이동시킵니다. 
				exus.OptionList.moveSelectedItem(id, id);

# moveItemUp

선택된 아이템을 위로 이동시킵니다. 
				exus.flash.generate(file, width, height, id);

# moveItemDown

선택된 아이템을 위로 이동시킵니다. 
				exus.flash.generate(file, width, height, id);

# moveItemTop

선택된 아이템을 가장 위로 이동시킵니다. 
				exus.flash.generate(file, width, height, id);

# moveItemBottom

선택된 아이템을 가장 아래로 이동시킵니다. 
				exus.flash.generate(file, width, height, id);
